## IHTC (NRA) - Trabalho final de Matemática Computacional

Este projeto utiliza o [uv](https://docs.astral.sh/uv/) como gerenciador de pacotes e ambientes virtuais, garantindo instalações ultrarrápidas e determinísticas.

### Pré-requisitos

Tenha o `uv` instalado na sua máquina e rode o comando `uv bash` para que o `uv` crie o ambiente virtual e instale todas as dependências.
